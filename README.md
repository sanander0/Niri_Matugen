## ⚙️ Софт

| Компонент              | Программа                  |
|------------------------|----------------------------|
| Терминал               | Alacritty                  |
| Панель                 | Waybar                     |
| Лаунчер                | Wofi                       |
| Цветовая схема         | Matugen                    |
| Уведомления            | Mako                       |
| Обои                   | Swww / Awww                |
| Сolor-PIcker           | Grim + Slurp + ImageMagisk |

## ⌨️ Бинды

| Клавиши        | Действие                    |
|----------------|-----------------------------|
| Mod + Q        | Открыть терминал            |
| Mod + W        | Лаунчер (wofi toggle)       |
| Super + B      | Firefox                     |
| Super + E      | Файловый менеджер (nemo)    |
| Super + T      | Telegram                    |
| Mod + Tab      | Overview                    |
| Mod + C        | Закрыть окно                |
| Mod + F        | Максимизировать             |
| Mod + V        | Floating режим              |
| Print          | Скриншот                    |
| F12            | Color picker                |


| Клавиши        | Действие              |
|----------------|-----------------------|
| Mod + H/J/K/L  | Перемещение фокуса    |
| Mod + ←↑↓→     | Перемещение фокуса    |
| Mod + Ctrl + H/J/K/L | Двигать окна    |
| Mod + Shift + H/J/K/L| Мониторы       |

### 🧱 Workspaces

| Клавиши        | Действие                     |
|----------------|------------------------------|
| Mod + 1..9     | Переключить workspace        |
| Mod + Shift + 1..9 | Переместить окно        |
| Mod + U / I    | Workspace вверх / вниз       |
